Baze Validation

Validate your form with ease

As per version 0.2.0, Baze Validation only validate if a field is empty but soon more validation will be added such as email and number validation.

Install

You can get the script via bower.

bower install baze-validation --save

It depends on jQuery so make sure include it after jQuery.

<script src="jquery.js"></script>
<script src="baze.validate.js"></script>

How to use

Baze Validation will validate any <form> element that has data-baze-validate attribute and required attribute on its children. If both conditions do not match, Baze will not validate the form.

<form data-baze-validate>
    <label for="username">Usernama</label>
    <input id="username" type="text" required>

    <button type="submit">Submit</button>
</form>

Then init the plugin by calling run function.

BazeValidate.run();

How it works

Baze Validation will listen on form submit event. When a form is submitted:

  1. Check for empty fields.
  2. If field is empty, class form-input--error will be added. A message will be placed after the field with class form-msg-error
  3. If field is not empty, class form-input--success will be added. No message will be added.

Baze validation has no default styling so it is up to you how you want to style it.

API

run()
Trigger Baze Validation.
setErrorClass( string )
set class name for invalid fields. default: form-input--error.
setSuccessClass( string )
set class name for valid fields. default: form-input--success.
setMsgClass( string )
set class name for form message. default: form-msg-error.
